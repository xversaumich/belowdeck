## Final requirements ##
## Visual: 
Your team should develop a web application using the basic concepts of HTML and CSS. Regarding HTML, it is up to the team which tags should be used. Related to CSS files, they need to be in an external file. (Complete)


## Bootstrap: 
To make your website more appealing, the team should use the Bootstrap framework for frontend. Basic elements should be included as Navbar, Containers, Columns, and Footer. Use the examplesLinks to an external site. provided by Bootstrap in case the team needs some inspirational content (Complete)

## Async Requests:
 You should load dynamically data from your backend by using AJAX+JSON (with GET and POST requests). Your javascript files should be added as external files into your HTML.


## Backend: 
Your team should develop the backend in Node.js and server all static files (.html, .css, .js, etc.).


## Execution:
 By running your Node.js app, your frontend and backend need to be up at the same time, for instance, only running node server.js.


## Features: 
You should deliver at least 3 features of your app. They will be determined by your team and your instructor. However, if your team is going to develop most of the features presented during the midterm, you don't need to talk with the instructor.


## Cloud Deployment: 
The team should deploy their app in the cloud. It means that the front end, back end, and database need to be deployed there. Talk with your instructor for more details and ask for some help (e.g. which cloud platform to use).
